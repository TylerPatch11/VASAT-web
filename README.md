# VASAT Client Website — V1 Source

This is a responsive single-page Next.js implementation of the V1 Vasat client website concept.

## Run it

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Structure

- `app/page.js` — complete V1 client-facing page
- `app/layout.js` — metadata and global layout
- `app/globals.css` — responsive design system and page styling
- `components/SiteHeader.jsx` — responsive navigation

## Important notes

- The Developers link points to `https://vasat.io/docs/`.
- The demo form is visual only in V1 and is not connected to a backend.
- Product claims and deployment details should be approved by the Vasat team before production.
- Replace the placeholder logo treatment with the official Vasat brand asset.
